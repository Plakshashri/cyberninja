## Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Plakshashri/cyberninja.git
   cd cyberninja
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```

4. **Visit**:  
   Navigate to `http://localhost:3000` in your browser.
